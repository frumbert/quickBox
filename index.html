quickBox - a no-frills modal overlay for jQuery.

There's not a lot of error handling or overhead in this plugin, or care for IE8 or below (as it should be in 2014). It's expected that you just want a simple way to show data in an overlay without all the hoo-ha you get with some alternatives.

  • Content from hidden div
    Loads content from an existing DOM object. Specify href='#div-id' on the link.
  • From file using href
    Loads content from an external file. Specify href='/path/to/file.htm' on the link.
  • Content from data-attributes
    Set content and other properties using data-attributes on the link, data-content='The content goes here'.
  • Using defaults in constructor
    Different display properties can be set from the constructor. $('a').quickBox({overlay:'#f0f'});
  • Image linked in href, width auto
    You can specify an image as the link href='/path/to/image.jpg'; we wait for the image to load first.
  • Embed a youtube video
    Stick the youtube iframe embed in the data-content attribute.

Settings

Settings can be set in the constructor, or over-ridden on individual links by using html5 data attributes. See the examples above for more information.

Default settings are:

{
	width: "auto",				// can be "auto", or "50%", or "50px", or "25em", etc
	height: "auto",				// can be "autp", or "75%", or "500px", or "5em", etc
	scrollable: false,			// if height is not "auto", then allows overflow-y to scroll
	escape: false,				// if you want to allow pressing escape to hide the box
	overlay: "#000"				// overlay is shown at 50% opacity of this colour
}
